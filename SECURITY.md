# Security Policy

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, use one of the following private channels:

- **Preferred:** GitHub's private [Security Advisories](https://github.com/myviliha/viliha-drizzle-crud/security/advisories/new)
  ("Report a vulnerability").

Please include:

- A description of the vulnerability and its impact
- Steps to reproduce (a minimal PoC if possible)
- Affected version(s)

We will acknowledge your report within **72 hours**, work with you on a fix, and
coordinate a disclosure timeline. Please give us reasonable time to release a fix
before any public disclosure.

## Supported Versions

Security fixes are provided for the latest released major version.
